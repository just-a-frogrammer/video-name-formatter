# video name formatter
 Simple tool to format videos titles written in C++ so they can be easily recognised by services such as Jellyfin.

 # Key features:
 1. Getting series season folder path from user
 2. Detecting episode and season number from filename
 3. Proposing changes for filenames
 4. Getting permission from user for changing filenames

 # Compilation:
 You compile it like any other C++ file, preferably using g++ compiler.

 # Usage:
 1. Launch program (if you want to use it on files owned by root you need to run it with sudo or from root account)
 2. Type in season's folder path do not use "" or ''
 3. check if changes proposed by program fit you
 4. confirm changes
 5. Congrats, you are finished
